À quoi ça sert
Repérer rapidement les ambassadeurs d'un lycée
Cette page est réservée aux administrateurs. Elle permet d'enregistrer uniquement les étudiants actuellement inscrits dans les écoles du groupe EDH avec leur lycée d'origine. Les inscriptions à un forum ne créent pas automatiquement un profil ici.
Bon usage
1. Ajouter les étudiants dès leur intégration
2. Renseigner précisément le lycée d'origine
3. Utiliser la recherche lycée pour identifier des profils mobilisables
